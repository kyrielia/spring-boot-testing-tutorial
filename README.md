# Test Kite Tutorial

An example project to demonstrate the usage of the _test kite_ on a Spring Boot microservice. The four testing tiers are:

- Unit Tests
- Application Tests
- Contract Tests
- E2E Tests

The different types of tests have been split into different Gradle source sets and Gradle tasks to make them easier to 
find and run. The same pattern should be applied when writing these tests for yourself, as it will keep things organised, 
and make it easier to run the tests that are relevant to your needs at the time.
